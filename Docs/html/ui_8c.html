void show_objects ( void  )

Scans the surrounding area and prints the objects that are found.

Initiates an IR scan of the surrounding area to find all objects of >1 degree width. The distance of the object, the angular location, and the calculated width of the object are sent over UART If in_program_ui is set, the output is in a machine readable format.
